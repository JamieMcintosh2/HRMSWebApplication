Technological Breakdown

Microservices

This application has been created using an emerging Software Architecture called Microservices. This is a form of Service Oriented Architecture (SOA) which is a collection of services representing a business process for the enterprise organisation, each of these services are components of the system and tend to expose their individual functionality over Rest APIs.

Microservices is an approach used to develop Service Oriented Architecture and takes the decoupling of services a step further, where each individual component of the system is broken down into a small independently deployable service which prioritises one single business process, these services do not need to communicate with other sections of the system and have their own individual database. This allows for great scalability and Fault Tolerance which this application demonstrates.

This application has split the HR Domain into 4 Microservices: Profile Service, Employment Service, Recruitment Service and Development Service. These Services have then been containerised using Docker and Deployed on Azure, for a completely Serverless application.

Generative Artificial Intelligence

At the end of 2022 OpenAI released ChatGPT to the public, revolutionizing the way we use Artificial Intelligence (AI)

The release of this technology resulted in professionals in the HR Domain suggesting that this technology could be used to help assist HR workers with some every day tasks their work requires.

Based on the suggestions of Professionals from the HR domain, this Project used the OpenAI API to Generate Job Adverts and Job Interview questions in a variety of spoken languages with great success allowing for improved efficiency and assisting large organisations with global recruitment.
